# ✍️ Automatic Copywriting and Tone Transformer

A powerful AI-powered tool that generates compelling copywriting content and transforms text to different tones using Google's Gemini AI.

## Features

- 🚀 **AI-Powered Copywriting**: Generate creative copy using Google Gemini AI
- 🎨 **Multiple Tone Options**: Professional, Casual, Persuasive, Informative, Emotional, Humorous
- 🔄 **Tone Transformation**: Change the tone of existing text instantly
- 📋 **One-Click Copy**: Copy generated content to clipboard
