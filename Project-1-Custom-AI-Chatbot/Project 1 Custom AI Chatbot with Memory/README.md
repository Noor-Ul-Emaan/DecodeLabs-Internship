# 🤖 Custom AI Chatbot with Memory

A terminal-based conversational AI chatbot that maintains an in-memory conversation history during a live session.

## Features

- 🧠 **In-Memory Conversation History**: Stores all messages in a list array
- 🔄 **Context Preservation**: Sends full history to LLM for contextual responses
- 💬 **Interactive Terminal**: Simple chat interface with commands
- 📊 **History Display**: View full conversation at any time
- 🗑️ **Clear Memory**: Reset conversation when needed
- 🚀 **Powered by Gemini**: Uses Google's Gemini 1.5 Flash model

## Commands

- **Type your message**: Chat with the bot
- **`history`**: View full conversation history
- **`clear`**: Clear conversation history
- **`exit`**: End the chat session
